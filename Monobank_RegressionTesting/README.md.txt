## Description

This project is aimed at improving skills in writing test cases for regression testing.

## Goal

This task will help me refine my skills in creating test cases for testing the functionality of mobile banking. Positive and negative test cases will ensure that changes in the program have not affected other parts of the program that were previously working correctly.

## What was done

Analyzed and reviewed requirements for Monobank mobile app functionality
Created Test cases
Regression Testing of Application
